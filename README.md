## VERSIONS 
Elixir 1.14.5
Erlang/OTP 26 

## USAGE
Start Shopping
```
Cashier.start()
```

Add items to Cart
```
Cashier.add_to_cart("GR1")
Cashier.add_to_cart("SR1")
```

View Cart
```
Cashier.view_cart()
```

Remove From Cart
```
Cashier.remove_from_cart("GR1")
```

Clear Cart
```
Cashier.clear_cart()
```

Checkout
```
Cashier.checkout()
```
